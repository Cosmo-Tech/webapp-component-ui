# EditableLabel

The aim of this component is to display an editable label whose value can be checked

## Props

<Props of={EditableLabel} />

## Basic Usage

<Playground>
  <EditableLabel
    value="Foo"
    onNewValue={() => console.log('New value accepted')}
    checkValue={(value) => {
      if (value.length === 0) {
        return 'TextField cannot be empty';
      }
      return null;
    }}
    labels={{
      title: 'TextField',
    }}
    typographyProps={{ variant: 'h4' }}
    textFieldProps={{ variant: 'outlined' }}
  />
</Playground>
