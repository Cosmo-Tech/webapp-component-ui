# Help Menu

The aim of this component is to display an interactive view of a list of errors

## Props

<Props of={ErrorsPanel} />

## Basic Usage

<Playground>
  <ErrorsPanel
    errors={[
      { summary: 'Error 1 summary', loc: 'someFile.js : line 1', context: 'Error 1 context' },
      { summary: 'Error 2 summary', loc: 'someFile.js : line 2', context: 'Error 2 context' }
    ]}
    onClear={() => {
      console.log('clear errors');
    }}
  />
</Playground>
