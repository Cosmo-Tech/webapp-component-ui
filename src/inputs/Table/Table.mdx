# Upload File

The aim of this component is to display a dataset CSV file in a table
and lets users edit its content.

## Props

<Props of={Table} />

## Basic Usage

<Playground>
  <Table
    editMode={true}
    dataStatus="READY"
    errors={[{ summary: 'Wrong type', loc: 'Line 3', context: 'Can\'t convert "foo" to int' }]}
    columns={[
      { field: 'a', type: ['string'] },
      { field: 'b', type: ['int'] },
      { field: 'c', type: ['bool'] }
    ]}
    rows={[
      { a: 'foo', b: '12', c: 'false' },
      { a: 'bar', b: '53', c: 'true' }
    ]}
    onCellChange={() => {
      console.log('cells changed');
    }}
    onClearErrors={() => {
      console.log('clear errors');
    }}
    buildErrorsPanelTitle={(errorsCount) => `${errorsCount} errors found during import:`}
  />
</Playground>
